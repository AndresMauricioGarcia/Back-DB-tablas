## Comandos
```cmd

# Crear entorno
py -3 -m venv .venv

# Activar entorno
.venv\Scripts\activate

# Guardar dependencias del entorno
pip freeze > requirements.txt

# Instalar dependencias desde un `requirements.txt`
pip install -r requirements.txt

```
